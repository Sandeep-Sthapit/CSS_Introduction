What is CSS?

  • CSS is Cascading Style Sheets.
  • Describes how to display HTML elements on the screen
  • HTML stands for Hyper Text Markup Language.
  • HTML describes structure of a web page.
  • If HTML is the skeleton to our body, then Cascading Style Sheets (CSS) is the skin.

ATTRIBUTES

ATTRIBUTES

  • id - unique identifier for an HTML element. Cannot be shared
  • class - categorize element into specific class. Can be shared
  • There are many other tags like href, src, name, etc.
  • The tags and attributes combine to create a webpage.

Selectors

CSS Selectors and Properties

  • Selectors are patterns to select one or more elements
  • Elements can be selected using tagname, classnames, ids, or attributes.
  • CSS Properties can change everything. It can change color, size, position, opacity, margin, border, etc.
